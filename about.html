About Our Project

The Smart Personal Resource Organizer is a web-based application designed to help users organize and manage their digital resources in one place. In today’s digital world, people often save important files, links, and information across different platforms such as messaging apps, screenshots, browser bookmarks, and social media.

This project provides a centralized platform where users can store and access important materials easily. The system allows users to upload and store PDF documents, save frequently visited website bookmarks, store links to online content such as videos or images, manage passwords, and record personal ideas or notes.

The application is built using HTML, CSS, and JavaScript to create a simple and interactive user interface. The main goal of the project is to improve digital organization and make it easier for users to retrieve saved information without spending time searching through multiple applications.
